# zombie-flush
A little and ugly python3 terminal game, all the code made by Santiago De Andrade 06/2021

Usage:

    ./(instalation_path)/main.py
    
Controls:
    
    moving :
      right   : d
      left    : a
      down    : s
      uo      : w
    
    shooting
      right : ;
      left  : k
      up    : o
      down  : l
    
    others:
      menu : 'space'
