# zombie-flush
A little and ugly python3 terminal game, all the code made by Santiago De Andrade 06/2021

Usage:

    cd (instalation_path)
    ./main.py
    
Controls:
    
    moving :
      right   : d
      left    : a
      down    : s
      up      : w
    
    shooting
      right : ;
      left  : k
      up    : o
      down  : l
    
    others:
      menu : 'space'
    
Recomendations: 
        
        1- Press f11 to activate full screen, the game was made considering that
        2- Flush as much zombies as you can !
